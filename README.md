FULL FEATURE LIST

### Admin Panel
- Secure login/logout
- Manage users: approve, block, delete accounts
- Manage job postings: approve / reject reported jobs
- Monitor job statuses (Pending / Ongoing / Completed)
- System reports: total users, active jobs, completed jobs

### Client Functions
- Register/login as a client
- Post jobs (title, description, category, budget, deadline)
- Browse all freelancer profiles and invite them to apply
- Review freelancer proposals and accept/reject them
- Track job status (Pending → Ongoing → Completed)
- Give ratings and feedback after job completion
- Messaging system with freelancers

### Freelancer Functions
- Register/login as a freelancer
- Create/update profile: skills, portfolio URL, hourly rate
- Browse available jobs (filter by category & budget)
- Apply for jobs: submit proposal with bid amount & description
- Track ongoing jobs
- Receive ratings and reviews after job completion
- Messaging system with clients
- Resume upload & AI skill extraction (PDFBox)

### Matching Algorithm
- Freelancer side: projects ranked by % skill match
- Client side: freelancers ranked by % skill match per project
